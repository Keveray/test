git commit (запомнить изменения)
git status (запросить статус текущих изменений)
git push (отправить изменения на удаленный репозиторий)
git pull (забрать изменения из удаленного репозитория)

git add (добавить текущие изменение в следующий commit)
git branch (создать новую ветку)
git merge (влить ветку в текущую)
git squash (объединить несколько commit в один)
git cherrypick (перенести commit с одной ветки на другую)
git checkout (перейти на другую ветку)
git rebase (обновить текущую ветку относительно другой)
git как откатить на предыдущую версию
